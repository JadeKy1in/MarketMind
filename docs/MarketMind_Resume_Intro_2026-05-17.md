# MarketMind — 投资分析平台项目介绍

**用途**: 简历/作品集 | **日期**: 2026-05-17

---

## 项目概述

MarketMind 是一个基于大语言模型的多智能体投资信号验证与决策支持平台。系统通过 21+ 独立 AI 影子分析师并行运行，每位分析师拥有独立的策略偏好与领域专长，在多指标复合排名下竞争决策权重，并内置合谋检测、挑战者淘汰、紧急配额审计等对抗性质量保障机制。平台覆盖 6 层信息源（新闻/社交媒体/市场数据/SEC 监管文件/宏观经济/期权与财报日历），经 3 层递进分析流水线（叙事层 - 基本面 - 技术面）产出结构化投资决策，辅以 3 级分层记忆系统与结晶引擎实现跨日知识积累，最终通过 CustomTkinter 指令中心提供实时的影子排名仪表盘与决策卡片界面。

## 核心技术成果

- 设计并实现 21+ 独立 AI 影子分析代理系统，涵盖 15 个领域专家、5 种高风险策略、合谋检测引擎、挑战者淘汰引擎等 7 种子代理类型，实现真正的多视角对抗性分析
- 构建 MPPM/Calmar/Omega/WR 四维复合排名引擎，结合贝叶斯置信度衰减与成就阶梯机制，自动对各影子分析师的历史表现进行加权排序
- 搭建 3 层递进分析流水线（叙事 - 基本面 5 级递进 - 技术面 3 灯审查），实现从定性到定量的结构化投资分析闭环
- 集成 6 层异构数据源（27 个新闻源、Bluesky/Reddit 社交媒体、SEC Form 4/13F 内部人交易、FRED/EIA/CFTC 宏观数据、期权流/经济日历），统一通过 httpx 异步网关接入
- 实现 3 级分层记忆架构（工作记忆 - 情景记忆 - 语义记忆）与结晶引擎，支持 Insights 自动提取、验证、升级/淘汰的跨会话知识积累
- 建立 Prompt 注入防御、API Key 轮换、M1 数据完整性协议、Token 预算限制等 4 层安全与完整性保障体系
- 开发 CustomTkinter 指令中心 GUI，包含实时影子排名仪表盘、趋势图表、决策卡片、门控面板等 10+ UI 组件

## 架构亮点

- **多智能体影子生态系统**：每种影子类型拥有独立的分析周期、虚拟投资组合与配额管理，通过 SQLite WAL 模式 10 表持久化全量状态，实现无共享并行分析
- **Gateway 模式 + M1 完整性注入**：所有 LLM 调用经统一网关路由（Flash/Pro 模型分流），网关层自动注入 M1 数据完整性协议，确保每次调用携带完整的上下文约束
- **对抗性质量保障闭环**：Catfish 少数派意见强制者（>=80% 共识触发）、Challenger 3 阶段盲审淘汰、Collusion 合谋检测（区分收敛与跟风）三重机制互为补充
- **方法论外置化与进化系统**：15 个专家 Prompt 从代码中解耦为 JSON 配置文件，配合 MethodologyEvolver 实现基于历史表现的方法论自动迭代
