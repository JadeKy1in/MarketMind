# 产品上下文

## 产品存在的理由（Why）

传统量化模型深陷"数据拟合"陷阱——在历史回测中表现完美，但在实盘样本外测试中因过度学习市场噪声而失效。华尔街的多因子挖掘范式忽略了金融市场本质上的非线性和反身性特征：投资者的认知与市场基本面是相互作用的双向反馈循环，而非单向反映关系。

本系统根植于乔治-索罗斯的反身性理论，采用"自上而下"的逻辑演绎替代纯数据驱动。核心信念：市场丰厚利润并非来自数据拟合，而是来自对宏观基本面的深度理解、对事件冲击链的精准预判、对技术结构的耐心跟随、以及对人性反身性情绪的冷静把握。

## 目标用户画像

- 独立宏观投资者，具备基本的金融认知框架
- 深信长期价值与逆向投资哲学
- 厌恶高频交易的摩擦成本与噪音消耗
- 追求每年仅 3-5 次高确定性机会，具有"不惧空仓"的耐心
- 通过 DeepSeek API 使用系统的决策辅助功能

## 用户旅程

```
每日盘前（用户触发）:
  -> 运行系统分析脚本
  -> 系统自动采集当日宏观事件、政要发言、市场行情
  -> Pro 模型（DeepSeek）执行四维共振逻辑推演
  -> 输出决策报告

用户评估:
  -> 阅读报告中的三个板块
     (1) 当前持仓处置建议（Hold/Sell）
     (2) 宏观与逻辑推演深度分析
     (3) 最终行动指令（Buy/Sell/保持空仓）
  -> 若系统建议买入: 手动对照 JSON 账户状态，人工判断仓位
  -> 若系统建议空仓: 严格遵守，不进行任何操作

每周复盘:
  -> 检查系统推演逻辑与市场实际走势的一致性
  -> 更新本地 JSON 账户状态文件
```

## 核心价值主张

- **低频高确定性**: 不追求交易频率，过滤掉 95% 以上的市场噪音
- **逻辑透明**: 每个交易信号附带完整的演绎推理链，而非黑箱信号
- **物理隔离安全**: 零券商 API 连接，账户状态人工管理，无安全风险
- **反拟合架构**: 不可通过历史数据反复调参来"优化"表现

## 与现有量化平台的差异化

| 维度 | 传统量化平台 | SignalFoundry |
|---|---|---|
| 方法论 | 因子挖掘、数据拟合 | 反身性理论、逻辑演绎 |
| 交易频率 | 高频/中频 | 极低频（年 3-5 次） |
| 信号依据 | 统计套利、量价模型 | 四维共振（基本面+技术+事件+情绪） |
| 执行方式 | API 直连自动交易 | 人工确认（系统输出报告） |
| 防过拟合 | Walk-forward, 交叉验证 | 逻辑链硬过滤 + 空仓压力测试 |
| LLM 后端 | 无 | DeepSeek（唯一指定） |
