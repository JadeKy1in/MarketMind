## 数据验证链：一条完整逻辑链示例

```
标题: "ECB keeps rates on hold in face of inflation threat"
         │ Flash评分: impact=7, macro, urgency=8
         ▼
主AI浏览 → 选择此线索
         │
期望差分析: 
  市场定价了"维持不变"吗？→ CME FedWatch: 90%概率维持 → |实际-预期|=10% < 15%
  → 标记 "mostly priced_in"，但通胀威胁措辞是新的 → 降优先级但继续
         │
HVR假设: "ECB通胀措辞鹰派 → 可能被迫提前加息 → EUR走强"
         │
Layer 1 (市场定价): EUR/USD期货定价 → 市场仅定价20%加息概率 → 支持假设 (0.8)
Layer 2 (基本面): FRED 欧洲CPI数据 → 核心CPI 3.2% 高于2%目标 → 支持 (0.9)
Layer 3 (多源): FT+Reuters+Bloomberg 3源都提到通胀措辞变化 → 支持 (0.85)
Layer 4 (历史): 2022年类似措辞后3个月EUR涨4% → 部分支持 (0.65)
         │
加权: 0.30×0.8 + 0.25×0.9 + 0.25×0.85 + 0.20×0.65 = 0.24+0.225+0.2125+0.13 = 0.8075
         │
判定: 0.8075 ≥ 0.70 且 Layer1+2+3 都有确认 → VERIFIED → 可形成投资建议
         │
对抗性自检: 强制看空论证"EUR可能不涨: ECB措辞可能只是口头干预,实际不加息"
         │ 看空置信度 0.45 < 0.60×0.8075=0.4845 → 不降级
         │
最终: EUR看涨, 置信度0.81, 进入Decision阶段
```
